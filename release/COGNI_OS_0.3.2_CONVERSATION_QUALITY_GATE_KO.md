# Cogni-OS v0.3.2 대화 품질 릴리스 게이트

## 수정 대상

v0.3.1에서 사용자가 입력한 다음 연속 질문은 자연스러운 대화 대신 고정된 품질
안전 문구로 끝났다.

1. `그럼, 나와함께 재미있는 프로젝트를 합시다.`
2. `나와 어떤 일을 함께 할 수 있나요?`

고정 문구가 예외를 숨기지는 않았지만 UI가 이를 `완료`로 표시했고, 형식 중심
검증은 이 일상적인 사용 흐름을 포함하지 않았다.

## v0.3.2 승인 기준

- 위 2턴을 같은 세션에서 원문 그대로 통과한다.
- 위 두 턴은 제품의 제한된 `conversation_fastpath`가 모델·Fact-book보다 먼저 한
  번만 응답한다. 일반 지식·코드·형식 요청은 이 경로가 가로채지 않는다.
- 독립 인사, 협업 제안, 바꿔 말하기, 오타, 후속 질문, 문맥 전환, 형식 요청 및
  Fact-book 회귀를 포함한 10턴 전체를 통과한다.
- 품질 fallback은 0회이며 사용자 요청 하나당 assistant 답변은 하나뿐이다.
- 답변은 반복·역할 누출·미완성 경계가 없고 턴별 120초 기본 상한을 지킨다.
- 문장 번호를 문장으로 오인하지 않고, 정확한 문장 수와 질문 주제 관련성을 함께
  충족한다.
- UI는 fallback을 `품질 검증 실패 · 복구 필요`로 표시하고 정상 완료로 위장하지
  않으며, 제한된 소셜 응답은 `대화 FAST PATH · 완료`로 구분한다.

실행법과 JSON 필드는 `docs/CASUAL_KOREAN_VALIDATION.md`를 따른다. 실제 로컬 Gemma
증거 JSON이 `strict_casual_gate_passed=true`가 되기 전에는 이 문서만으로 수정
완료를 주장할 수 없다.
